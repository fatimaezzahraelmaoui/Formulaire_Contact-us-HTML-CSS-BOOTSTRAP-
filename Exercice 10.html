langage de marquage hypertexte (HyperText Markup Language en anglais). Il s'agit du langage de base du Web (World Wide Web). HTML n'est pas un langage de programmation proprement dit, mais plutôt un code de marquage. Il permet de décrire la page Web élément par élément servant de balises de description. Le code HTML est déchiffré par le navigateur du client. Tous les navigateurs existant reconnaissent parfaitement la syntaxe HTML. Les pages web faites uniquement avec HTML sont suffixées par l'extension. htm ou encore .html. L'édition d'une page web en code HTML est tellement .
